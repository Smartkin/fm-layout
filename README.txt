Installation:
* From the obs-installation-folder copy everything EXCEPT the obs-plugins folder into the OBS's executable directory.
* Copy over the files from obs-plugins directory into obs-plugins/64bit
* In OBS itself go to Tools->Scripts. Click the "+" icon and select init.lua file.
* It should load in the script and then press "Create layouts" button to create all the necessary scenes.
* Other instructions will be present when you load in the script.
